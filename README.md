# 🌐 TransLingua: AI-Powered Multi-Language Translator

TransLingua is a web-based language translation application built using
Streamlit and Google Gemini AI. It enables users to translate text
between multiple languages with high accuracy and contextual relevance.

---

## 📌 Features

- 🌍 Supports multiple languages
- 🤖 AI-powered translation using Gemini models
- ⚡ Real-time translation
- 🖥️ Simple and user-friendly interface
- 💾 Secure API integration
- 🚀 Fast and responsive performance

---

## 🛠️ Technologies Used

- Python 3.10+
- Streamlit
- Google Gemini API

- Generative AI Models

---

## 🔑 API Key Setup

1. Visit: https://makersuite.google.com/app/apikey
2. Generate a new API key
3. Create a `.env` file in the project folder
4. Add the following line:
GOOGLE_API_KEY=your_api_key_here

Replace `your_api_key_here` with your actual API key.
